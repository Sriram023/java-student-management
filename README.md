# Smart Student Management System

## Overview

The Smart Student Management System is a web-based application developed using Java Spring Boot, Thymeleaf, and SQLite. The system helps manage student records efficiently through CRUD (Create, Read, Update, Delete) operations with a clean and responsive user interface.

---

# Features

* Add new student records
* View all student details
* Delete student records
* Responsive user interface using Bootstrap
* Database integration using SQLite
* MVC architecture implementation

# How to Run the Project

## Clone Repository

```bash
git clone https://github.com/your-username/smart-student-management-system-java.git
```

---

## Open Project

Open the project using:

* VS Code
* IntelliJ IDEA

---

## Run Application

```bash
mvnw.cmd spring-boot:run
```

---

## Open Browser

```plaintext

```
